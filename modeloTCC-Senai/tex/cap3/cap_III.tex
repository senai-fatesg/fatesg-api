\chapter{Ambiente de Produção}
\label{cap:ambienteProducao}

Para o funcionamento do sistema é recomendado que todo o ambiente se adeque aos seguintes requisitos mínimos:
\begin{itemize}
\item Servidor
	 \begin{itemize}
		  \item Requisitos mínimos
		  \begin{itemize}
		    \item HP ML310E
		    \item Processador:Intel Xeon E3-1220   
		    \item Espaço em disco: 1 TB
		    \item Memória RAM: 8.0 GB
		  \end{itemize}
		  
		  \item Software
		  \begin{itemize}
		    \item Servidor de Aplicação: Apache Tomcat 6.0
		    \item Banco de dados: PostgreSQL 
		  \end{itemize}		 
	\end{itemize}
	     
\item Infra estrutura
		\begin{itemize}
		  \item Rede TCP/IP estruturada 100/1000 GB	  
		  \end{itemize}
		  
\item Cliente
	 \begin{itemize}
		  \item Requisitos mínimos
		  \begin{itemize}
		    \item Processador: Intel i3 1.3 GHz
		    \item Memória RAM: 4.0 GB
		    \item Browser
		    \begin{itemize}
		   	 	\item Firefox: versão 29.0 ou maior
		   	 	\item Safari: versão 5.1 ou maior
		    	\item Opera: versão 14.0 ou maior
		    	\item Google Chrome: versão 35.0 ou maior
		    	\item Internet Explorer: versão 9.0 ou maior
		    \end{itemize}
		  \end{itemize} 		  
	  \end{itemize}
\end{itemize}
